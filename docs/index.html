Declassified

Privacy

Images are sent to the Mistral AI API for analysis and are not stored by this bot.

Player stats (kills, deaths, samples, etc.) extracted from screenshots are stored to power the /stats command.

Discord user ID and username are stored to link reports to players. No other personal data is collected.

The bot currently uses Mistral's Experiment plan: API requests may be used by Mistral to improve their models. With enough community support, we will upgrade to a paid plan where your data is no longer used for training.

Use the /about command in Discord for full details.
